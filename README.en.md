# 小象数字营销云平台--基于SDK的用户行为日志采集

#### Description
通过Nginx搭建接口服务器，接收终端SDK埋点上行报文，flume从Nginx日志中读取埋点数据后，进行解密和处理。将明文数据存放到kafka的Topic下，kafka将埋点数据存入数据库的处理。
